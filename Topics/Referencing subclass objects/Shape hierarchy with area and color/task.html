Shape hierarchy with area and color

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Create a class hierarchy for a basic shape system. The base class should be an abstract class called Shape with a protected instance variable for color. Extend the Shape class to create a Circle class that adds a radius variable, and a Rectangle class that adds length and width variables. Override a method called getArea in both subclasses to calculate and return the area of the shape. In main, create an instance of each subclass, set the color and dimensions, and print the area and color for each shape.
